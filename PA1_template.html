Are there differences in activity patterns between weekdays and weekends?

  • Create a new factor variable in the dataset with two levels  weekday and weekend indicating whether a given date is a weekday or weekend day.
cleandata["daytype"] <- ifelse((weekdays(as.Date(strptime(cleandata$date, format = "%Y-%m-%d"))) == 
    "Saturday" | weekdays(as.Date(strptime(cleandata$date, format = "%Y-%m-%d"))) == 
    "Sunday"), "weekend", "weekday")

daytype is the new factor variable. Now, original date contains the additional column. We are using the cleandata for the last plot as NAs will not allow the last plot to be drawn.
